# 🗂️ Task-manager

Gerenciador de tarefas simples e eficiente para organizar seu dia a dia. Este projeto foi desenvolvido com foco em praticidade, interface intuitiva e fácil integração com recursos web.

## 🚀 Funcionalidades

- ✅ Adicionar, editar e excluir tarefas
- 📌 Marcar tarefas como concluídas
- 🔍 Filtrar tarefas por status
- 🎧 Suporte a áudios (ex: lembretes sonoros)
- 💾 Armazenamento local com persistência

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- [Node.js](https://nodejs.org/) (se aplicável)
- Web Manifest (PWA-ready)
